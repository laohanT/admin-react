# 用户表 admin_users

```
id        无意义   bigint UNSIGNED primary key auto_increment   bigint 非负数 主键 自增  
username  账户名称 VACHAR(255) unique	 not null  唯一 不为空      ---rsa加密
password  账户密码 VACHAR(255) not null                            ---rsa加密
role_id   角色id  int unsigend not null foreign key   整数 非负数  不为空  外键   
status    状态    TINYINT    mysql中最小的整数类型，用户存储小范围的整数值  取值范围 -128 -- 127 或者  0-255
expend    扩展    TEXT  	
create_time 创建时间 timestamp default current_timestamp
update_time 更新时间 timestamp default curent_timestamp
operater   操作员   varchar(50)
username_hash 账户名称的hash值， 索引用于查询（rsa加密后的数据不能做匹配）
// 定义外键约束
foreign key (`role_id`) references `admin_roles`(`id`)
 on delete restrict
 on update cascade
```
